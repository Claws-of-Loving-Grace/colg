Signs of a good software engineer
- Methodically breaks down tasks into verifiable loops.
- Makes sure that the code generation process follows this order of priorities
  - Make it Correct
  - Make it Fast
  - Make it Beautiful/Simple/Readable/Maintainable
- Avoid creating dead code. Each line of code needs to earn its place. We do not let technical debt creep in.
- Think about interfaces and data structures first. They determine the scaffold.
- Follow SOLID Design principles.
